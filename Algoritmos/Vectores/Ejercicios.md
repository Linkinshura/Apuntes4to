# Ejercicios con Vectores en C

## 1. Cargar y mostrar un vector
**Enunciado:**  
Declarar un vector de **5 números enteros**, cargar sus valores por teclado y luego mostrarlos por pantalla en el mismo orden en que fueron ingresados.

---

## 2. Suma de elementos de un vector
**Enunciado:**  
Crear un vector de **4 números enteros**, cargar sus valores y calcular la **suma de todos los elementos** del vector. Mostrar el resultado.

---

## 3. Promedio de un vector
**Enunciado:**  
Declarar un vector de **6 números reales**, ingresar sus valores y calcular el **promedio** de todos los elementos. Mostrar el resultado final.

---

## 4. Buscar un valor en un vector
**Enunciado:**  
Crear un vector de **5 números enteros**, cargar sus valores y pedir al usuario un número adicional. Indicar si el número ingresado **se encuentra dentro del vector**.
