# Constantes y variables en JavaScript

En JavaScript se utilizan **variables** y **constantes** para almacenar datos que se usan en el programa.

## Variables

Una variable es un espacio en memoria que puede **cambiar su valor** durante la ejecuci√≥n del programa.

Se declaran con las palabras clave `let` o `var`.

```javascript
let nombre = "Juan";
var edad = 20;

nombre = "Pedro"; // Se puede cambiar
```          



## Constantes

Las constantes se declaran con const y no pueden cambiar su valor una vez asignadas.

```javascript
const PI = 3.1416;
const mensaje = "Hola Mundo"
```
