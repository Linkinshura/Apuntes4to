# Introducción a JavaScript y el DOM

## 1. Constantes y variables en JavaScript

En JavaScript se utilizan **variables** y **constantes** para almacenar datos que se usan en el programa.

### Variables

Una variable es un espacio en memoria que puede **cambiar su valor** durante la ejecución del programa.

Se declaran con las palabras clave `let` o `var`.

```javascript
let nombre = "Juan";
var edad = 20;

nombre = "Pedro"; // Se puede cambiar
```          



## Constantes

Las constantes se declaran con const y no pueden cambiar su valor una vez asignadas.

```javascript
const PI = 3.1416;
const mensaje = "Hola Mundo"
```


## Eventos

Los eventos son acciones que ocurren en la página, como hacer clic en un botón, mover el ratón, escribir en un campo o cargar el documento.

JavaScript puede escuchar estos eventos y responder ejecutando funciones.

```javascript
<button onclick="saludar()">Saludar</button>

<script>
  function saludar() {
    alert("Hola desde JavaScript");
  }
</script>
```

## DOM

El DOM (Document Object Model) es una representación del documento HTML en forma de árbol de objetos.

Este HTML se ve en el DOM:

```html
<body>
  <h1 id="titulo">Bienvenidos</h1>
  <p class="texto">Esto es un párrafo.</p>
</body>
```

Document
 └── html
      └── body
           ├── h1 (id="titulo")
           └── p (class="texto")
